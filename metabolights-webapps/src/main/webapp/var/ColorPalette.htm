The Corporate Fonts of EMBL

For standard publications, we recom- mend using the following four basic styles and weights: 55 New Helvetica Roman, 56 New Helvetica Italic, 75 New Helvetica Bold and 76 New Helvetica Bold Italic.

For more extensive use: 45 New Helvetica Light, 45 New Helvetica Light Italic, 65 New Helvetica Medium, 66 New Helvetica Medium Italic, 85 New Helvetica Heavy, 86 Helvetica Heavy Italic, 95 New Helvetica Black and 96 New Helvetica Black Italic.

In addition, all above mentioned fonts may be applied as “condensed” fonts.

If, for technical reasons, Helvetica should not be available, Arial (Microsoft) may be used by way of exception. As current part of Windows OS, this font is at the disposal of the majority of users.

EMBL Colours Guide

The primary colours defined for EMBL result from the colour components of the logotype. Colours are defined within different colour systems to be used according to any chosen application. Generally, a distinction is to be made between colour represented on printed matter or display monitor.

For offset printing, colour systems are, for example, four colour printing (CMYK in compliance with ISO 2846) or solid / spot colours (HKS, PANTONE).

For exclusive display on monitor either offline (e.g. Powerpoint by Microsoft) or online (e.g. world wide web browsers), colours are defined in compliance with the RGB system.
In addition, special colours for silk- screen printing, painting and plotter transparencies are defined, but not listed here.
